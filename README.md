🔥 Quora Clone

A modern and simple Quora-like web application built using HTML, CSS, and JavaScript. This project is created for learning front-end development and practicing DOM manipulation.

🚀 Features
📝 Create posts (questions/answers style)
💬 Interactive UI design
🎨 Clean and modern layout
⚡ Lightweight and fast
📱 Responsive design (mobile friendly)
🛠️ Tech Stack
HTML5
CSS3
JavaScript (Vanilla JS)
🎯 Purpose

This project is built for practice and learning purposes to improve front-end development skills, especially DOM manipulation and UI design.
